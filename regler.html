Så här spelas schack

Schack spelas på ett bräde med 64 rutor, varannan ruta svart och vit. Varje spelare börjar med 16 pjäser: kung, dam, två torn, två löpare, två hästar och åtta bönder. Målet är att sätta motståndarens kung i schackmatt.

Hur pjäserna rör sig:

  • Kungen ♔: ett steg åt vilket håll som helst
  • Damen ♕: hur långt som helst i alla riktningar
  • Torn ♖ (2x): rakt fram, bakåt eller åt sidan
  • Löpare ♗(2x): diagonalt hur långt som helst
  • Häst / Springare ♘ (2x) : går i ett "L"-mönster
  • Bonde ♙ (8x): ett steg framåt (två steg på första draget), slår diagonalt och kan göra ett speciellt drag som kallas en passant

Vinst

Spelet slutar när motståndarens kung inte kan undvika schack – det kallas schackmatt.
ELLER så kan spelet vinnas på tid där i ett parti med tid som faktor (Blitz, bullet, classical osv.) får motståndaren slut på sin tid och du vinner. Partiet kan också sluta oavgjort (remi) på flera olika sätt.
